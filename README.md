This project is a smart hospital appointment system designed to reduce patient waiting time and improve how doctors manage their schedules.

characteristics
   Patients can register and book appointments
   Doctors can see and manage their queue
   Only patients who have arrived are considered in the active queue
   The system automatically decides who should go next
   It also estimates how long patients will need to wait

Key Features
🔹 Smart Queue System
Patients are ordered based on:
Priority level
Appointment time
Only Arrived patients are included (real-time behavior)

🔹 AI-Based Time Prediction
The system predicts consultation time using:
Visit type (new or follow-up)
Patient visit history
Doctor’s average consultation time
Priority level

🔹 Live Wait Time
Calculates total waiting time dynamically
Updates automatically when queue changes

🔹 Notifications
Patients receive updates like:
Appointment booked
“You are next” alerts

Tech Stack
Backend: Python (Flask)
Database: MySQL
Frontend: HTML, CSS, JavaScript
